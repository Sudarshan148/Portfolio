Key Projects

Firewall Configuration: Dual-Path, Dual-Layer Architecture

This extensive project involved the meticulous configuration of Cisco switches and the optimization of critical network maintenance protocols to establish a highly resilient and secure network infrastructure. A core achievement was the proposal and successful implementation of an Active/Passive high-availability firewall cluster, ensuring continuous uptime and peak performance for enterprise operations. This involved advanced configurations of HSRP (Hot Standby Router Protocol) on Cisco routers and firewalls, meticulously setting up active and standby units for automatic, seamless failover in the event of a primary device failure.

I designed and deployed a robust network security framework specifically for Joe's Snacks, incorporating a sophisticated Dual-Path, Dual-Layer configuration. This architecture segregated traffic into distinct internal and external paths, complemented by multiple security layers, significantly enhancing the overall security posture and manageability of the network. The design focused on minimizing single points of failure, implementing redundant links, and providing deep packet inspection capabilities at various network segments.

Key infrastructure components built during this project included:

  • Secure WAN connectivity via encrypted VPN tunnels, ensuring safe data transmission across geographically dispersed locations.
  • Hardened sales servers, achieved through the application of least privilege access principles, regular security patching, and comprehensive vulnerability assessments.
  • Resilient HR systems, by implementing Network Access Control (NAC) to authenticate and authorize devices connecting to the HR network, alongside finely-tuned access control lists (ACLs) to restrict sensitive data access.
This project comprehensively demonstrated my proficiency in end-to-end network solution deployment, advanced security hardening, and high-availability design principles critical for business continuity.

Enterprise Network Design - Radeon Company Ltd.

As part of this capstone project, I architected a comprehensive and scalable hierarchical network solution tailored for a new four-story office building, strictly adhering to Cisco's recommended best practices for enterprise network design. The design meticulously planned and incorporated detailed VLAN segmentation (VLANs 10-120) across all floors and various departments (e.g., HR, Finance, IT, Guest networks) to logically separate and secure departmental traffic. This segmentation was crucial for preventing unauthorized cross-departmental access and significantly reducing broadcast domains, improving network efficiency.

Advanced Cisco switching technologies (specifically, Catalyst series switches at the access, distribution, and core layers) were strategically leveraged to implement these VLANs efficiently. This included the intelligent use of VTP (VLAN Trunking Protocol) for simplified VLAN management across the network infrastructure. Furthermore, I successfully deployed OSPF (Open Shortest Path First) as the interior gateway protocol, ensuring efficient, dynamic, and resilient inter-VLAN communication. OSPF's fast convergence properties were critical in boosting overall LAN performance and enhancing network resilience by quickly rerouting traffic in case of link failures.

The entire network design was rigorously validated and simulated using Cisco Modeling Labs (CML), a powerful network simulation tool. This virtual environment allowed for extensive testing of routing protocols (OSPF convergence, route redistribution), Quality of Service (QoS) policies (prioritizing voice and video traffic), and various security configurations (ACLs, port security) in a controlled setting. This thorough validation process ensured successful large-scale implementation and guaranteed optimal operational stability and performance before any physical deployment, mitigating potential risks and issues.

Jarvis Voice Assistant (Python & Speech Recognition)

This project involved the development of a multifunctional voice assistant, affectionately named "Jarvis," built entirely using Python scripting. The core functionality relied on seamless integration with Google Speech Recognition APIs for advanced natural language processing, allowing users to interact with the system using spoken commands. While primarily a software development project, foundational Cisco networking knowledge was applied to conceptualize and design how such an assistant could hypothetically enhance network monitoring and interaction capabilities within a simulated network environment, for instance, by voice-commanding diagnostics or fetching network status.

The project heavily utilized key Python libraries:

  • `pyttsx3`: Employed for high-quality, offline text-to-speech conversion, providing vocal feedback to the user.
  • `SpeechRecognition`: Integrated for accurate voice input processing, converting spoken words into actionable text commands.
  • Other libraries for specific tasks like `smtplib` for email dispatch, `webbrowser` for application launching, and `requests` for fetching real-time weather data by integrating with external APIs (e.g., OpenWeatherMap).

A significant focus was placed on implementing thorough documentation adhering to industry best practices, including detailed code comments, README files, and setup instructions. Rigorous unit and integration testing procedures were also put in place to ensure the system's reliability, stability, and robustness against various inputs and edge cases. This comprehensive approach ensured that Jarvis could perform seamless voice-driven tasks such as initiating email dispatch, launching specific network monitoring applications (like Wireshark or Cisco Packet Tracer in a simulated context), and providing real-time weather reports, showcasing strong Python development and API integration skills. This project culminated in a publication in the *International Journal of Innovative Research in Technology*.

Dual Dataset Predictive Modeling - COVID-19 & Titanic

In this analytical project, I applied advanced machine learning techniques to build predictive models based on two distinct and complex datasets: COVID-19 patient data and historical Titanic passenger data. The primary objective was to generate actionable insights and predict outcomes. For the COVID-19 dataset, the focus was on predicting health outcomes (e.g., recovery vs. mortality) based on patient symptoms, demographics, and comorbidities. For the Titanic disaster dataset, the goal was to predict historical survival (e.g., survived vs. perished) based on features like age, class, and sex.

Key methodologies and activities included:

  • Comprehensive Data Preprocessing: This involved meticulous handling of missing values (e.g., imputation techniques), feature scaling (standardization/normalization), and encoding categorical data (one-hot encoding) to prepare the datasets for model training.
  • Robust Model Training: I experimented with and trained various classification algorithms from the `scikit-learn` library, including Logistic Regression, Decision Trees, and Random Forests, selecting the most appropriate model for each dataset based on performance metrics.
  • Rigorous Cross-Validation: K-fold cross-validation was extensively used to ensure the models' generalization capability and prevent overfitting.
  • Performance Metric Evaluation: Models were thoroughly evaluated using a suite of performance metrics such as accuracy, precision, recall, and F1-score, providing a holistic view of their effectiveness.

These demonstrated analytical and predictive modeling skills are highly transferable and valuable across various IT roles, particularly in data-driven environments such as manufacturing and plant operations. They can be applied to diverse tasks like predictive maintenance of industrial equipment, anomaly detection in operational data, or optimizing resource allocation, enabling data-driven decision-making and significant operational optimization. This project highlights my ability to extract meaningful insights from complex data and build practical predictive solutions.
